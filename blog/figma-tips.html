Figma Tips · أسرار فيجما

Introduction · مقدمة

English: Figma is the industry standard for UI/UX design. But are you using it to its full potential? These tips will boost your productivity and design quality. From keyboard shortcuts to advanced component systems, mastering these techniques will cut your design time in half and elevate the consistency of your work.

العربية: فيجما هو المعيار الصناعي لتصميم واجهات المستخدم. لكن هل تستخدمه بأقصى إمكانياته؟ هذه النصائح سترفع إنتاجيتك وجودة تصميمك. من اختصارات لوحة المفاتيح إلى أنظمة المكونات المتقدمة.

Auto Layout · التخطيط التلقائي

English: Auto Layout is Figma's most powerful feature. It creates responsive containers that adjust automatically to content changes. Master the key properties: padding (individual sides or symmetric), spacing between items (horizontal and vertical gap), direction (horizontal row or vertical column), alignment (start, center, end, space-between), and sizing (fixed, hug contents, fill container). Use "hug" for buttons and tags, "fill" for full-width elements, "fixed" for explicit dimensions.

English: Nest Auto Layout frames for complex layouts: a card frame (vertical auto layout) containing an image (fill width, fixed height), a text block (vertical auto layout with padding), and a button row (horizontal auto layout with space-between). Each inner frame has its own Auto Layout settings. This creates a fully responsive component tree where changing one element reflows everything automatically. Use absolute positioning within Auto Layout for badges, icons, and overlays that shouldn't affect the flow.

العربية: التخطيط التلقائي هو أقوى ميزة في فيجما. ينشئ حاويات متجاوبة تتكيف تلقائيًا مع المحتوى. أتقن الحشو والتباعد والاتجاه والمحاذاة والتحجيم. استخدم "hug" للأزرار، "fill" للعناصر كاملة العرض. كدس إطارات Auto Layout للتخطيطات المعقدة — إطار بطاقة يحتوي على صورة ونص وزر، كل بإعدادات تخطيط تلقائي خاصة به.

Components & Variants · المكونات والمتغيرات

English: Create master components with variants for different states (default, hover, active, disabled, loading, error, focused). Use component properties to expose customizable options: text (swap text instances), boolean (show/hide elements), instance swap (swap between icon sets), and variant (switch between style variants like primary/secondary/outline/ghost buttons in sizes sm/md/lg). A well-structured component library reduces design time by 60-80%.

English: Organize components using clear naming conventions with forward slashes for nesting: "Button / Primary / Medium", "Button / Secondary / Small", "Input / Text / Default". This creates a hierarchical dropdown in the assets panel. Use team libraries (publish as a team library) to share components across multiple files. When you update a master component, all instances across every file update automatically — this is the single source of truth for your design system.

العربية: أنشئ مكونات رئيسية بمتغيرات للحالات المختلفة (مبدئي، hover، نشط، معطل، خطأ). استخدم خصائص المكونات لتبديل النصوص والأيقونات والرؤية. نظم المكونات بتسميات واضحة: "زر / أساسي / متوسط". استخدم مكتبات الفريق لمشاركة المكونات عبر ملفات متعددة.

Keyboard Shortcuts · اختصارات لوحة المفاتيح

English: Essential shortcuts every Figma user must know: Ctrl+D — duplicate (or Alt+Drag), Ctrl+G — group selection, Ctrl+Shift+G — ungroup, Ctrl+[ and Ctrl+] — move layer backward/forward, Ctrl+Shift+[ and Ctrl+Shift+] — send to back/front. Tool shortcuts: R — rectangle, O — oval, T — text, L — line, P — pen, Shift+1 — fit to screen, Shift+2 — zoom to selection.

English: Advanced shortcuts: Ctrl+Alt+C — copy properties, Ctrl+Alt+V — paste properties (like format painter). Ctrl+Shift+E — flatten selection to image. Alt — show measurement distances between layers. Hold Ctrl while dragging to override Auto Layout constraints. Double-click any layer to enter isolation mode. / — search anything in the file. Ctrl+P — quick search commands. Learning these shortcuts will save hours every week.

العربية: الاختصارات الأساسية: Ctrl+D — نسخ. Ctrl+G — مجموعة. R — مستطيل. O — بيضاوي. T — نص. Shift+1 — ملء الشاشة. اختصارات متقدمة: Ctrl+Alt+C/V — نسخ/لصق الخصائص. Alt — إظهار مسافات القياس. / — بحث أي شيء. إتقان هذه الاختصارات سيوفر ساعات كل أسبوع.

Design Systems & Variables · أنظمة التصميم والمتغيرات

English: Use Figma Variables (introduced 2024) for theme management. Create primitive variables for raw values (--blue-500: #3B82F6), then alias them as semantic tokens (--color-primary: --blue-500). Create modes for light and dark themes. Link component properties to variables so theme switching happens automatically across all components. This is the professional way to maintain multiple brand themes in a single file.

English: Structure your design system with dedicated pages: a "Design System" page containing all color styles, text styles, effect styles, variables, and component libraries. Use "Cover" pages for presentation. Create a "Sandbox" page for experimentation. Keep an "Archive" page for deprecated elements. This structure keeps your file organized and makes it easy for team members to find what they need without creating chaos.

العربية: استخدم متغيرات فيجما (2024) لإدارة السمات. أنشئ متغيرات أولية للقيم الخام ثم سمها كرموز دلالية. أنشئ أنماطًا للسمات الفاتحة والداكنة. نظم ملفك بصفحات مخصصة: صفحة "نظام تصميم" تحتوي على جميع الأنماط والمكونات. هذا يبقي ملفك منظمًا.

Plugins · الإضافات

English: Top Figma plugins for productivity: Content Reel (dummy text, images, avatars, flags), Iconify (100,000+ open-source icons accessible instantly), Stark (accessibility — contrast checking, colorblind simulation), Remove BG (AI-powered background removal), FigJam AI (diagram and flow generation), Autoflow (connect user flows with arrows), Figmotion (create animations inside Figma), and Unsplash (free stock photos directly in canvas).

English: For specialized workflows: Anima (convert Figma to React/Vue code), Zeplin or Avocode (developer handoff), LottieFiles (import and export Lottie animations), Chart (data visualization charts), Noise & Texture (add realistic texture), and Broken Link Checker (find disconnected prototypes). Only install plugins you actually use — too many plugins slow down Figma's performance. Review your plugins quarterly and remove unused ones.

العربية: أفضل إضافات فيجما: Content Reel (نصوص وصور وهمية)، Iconify (مجموعات أيقونات)، Stark (إمكانية الوصول)، Remove BG (خلفيات الصور)، Anima (تحويل التصميم لكود)، LottieFiles (رسوم متحركة). راجع إضافاتك كل 3 أشهر وأزل غير المستخدمة منها.

File Organization · تنظيم الملفات

English: A clean file is a fast file. Use consistent naming conventions: "Page Name | Variant — Description". Name all layers meaningfully — never leave default names like "Frame 1" or "Rectangle 2". Use sections (Shift+A) to group related frames. Use pages to separate distinct workstreams. Hide or delete unused layers. Each layer should have a purpose — if it doesn't, remove it.

English: Cover pages create a professional impression. Name your pages clearly: "Cover", "Design System", "Screens — Home", "Screens — Onboarding", "Screens — Settings", "Components", "Prototype Flows", "Archived". Use the "✕" component (not the layer visibility toggle) to reduce clutter in the layers panel. A well-organized Figma file is respected by teammates and makes collaboration seamless.

العربية: نظم ملفاتك بتسميات متسقة: "اسم الصفحة | النوع — الوصف". سم جميع الطبقات بشكل ذي معنى. استخدم الأقسام لتجميع الإطارات المرتبطة. استخدم الصفحات لفصل مسارات العمل المختلفة. أخفِ أو احذف الطبقات غير المستخدمة. صفحة غلاف احترافية تترك انطباعًا جيدًا لدى العملاء والزملاء.
