DaVinci Resolve Guide · دليل دافنشي ريزولف

25 min read · 25 دقيقة

Introduction · مقدمة

English: DaVinci Resolve is the industry's most powerful free video editor. Developed by Blackmagic Design, it combines professional editing, color grading, visual effects, motion graphics, and audio post-production in a single integrated application. Hollywood blockbusters, Netflix series, YouTube creators, and broadcast studios all rely on Resolve for its unmatched color science and node-based workflow. The free version is remarkably capable — it lacks only a few advanced features like noise reduction, 4K export above 60fps, and a handful of effects. For almost all content creation, the free version is more than sufficient. This comprehensive guide covers the full Resolve workflow from import to export.

العربية: دافنشي ريزولف هو أقوى محرر فيديو مجاني في الصناعة. طورته Blackmagic Design، يجمع بين المونتاج الاحترافي وتصحيح الألوان والمؤثرات البصرية والرسوم المتحركة والصوت في تطبيق واحد متكامل. أفلام هوليوود الضخمة ومسلسلات Netflix ومنشئو YouTube واستوديوهات البث جميعهم يعتمدون على Resolve لعلوم الألوان غير المسبوقة وسير العمل القائم على العقد. النسخة المجانية قوية بشكل ملحوظ — تفتقد فقط لبعض الميزات المتقدمة مثل تقليل الضوضاء والتصدير بدقة 4K فوق 60 إطارًا في الثانية. لجميع أنواع إنشاء المحتوى تقريبًا، النسخة المجانية أكثر من كافية. هذا الدليل الشامل يغطي سير عمل Resolve الكامل من الاستيراد إلى التصدير.

The Interface · الواجهة

English: Resolve has seven dedicated workspaces, each designed for a specific task. You switch between them using the buttons at the bottom of the screen. The Cut page is optimized for fast editing with dual timelines and source tape mode — perfect for quick turnaround projects like social media content. The Edit page provides a full-featured timeline with up to 99 video and audio tracks, advanced trimming tools, and transition controls. The Fusion page is a node-based compositing environment for visual effects and motion graphics. The Color page is where Resolve truly shines — it offers the most advanced color grading tools in the industry. The Fairlight page is a full digital audio workstation for sound design, mixing, and mastering. The Deliver page handles export with dozens of format presets. There is also the Media page for importing, organizing, and managing footage.

العربية: دافنشي لديه سبع مساحات عمل مخصصة، كل منها مصممة لمهمة محددة. تتنقل بينها باستخدام الأزرار في أسفل الشاشة. صفحة Cut محسّنة للتحرير السريع بخطين زمنيين — مثالية للمشاريع سريعة التسليم مثل محتوى وسائل التواصل. صفحة Edit توفر خطًا زمنيًا كامل الميزات مع ما يصل إلى 99 مسار فيديو وصوت وأدوات قص متقدمة. صفحة Fusion هي بيئة تركيب قائمة على العقد للمؤثرات البصرية والرسوم المتحركة. صفحة Color هي حيث يتألق Resolve حقًا — تقدم أكثر أدوات تصحيح الألوان تقدمًا في الصناعة. صفحة Fairlight هي محطة عمل صوتية رقمية كاملة لتصميم الصوت والمزج. صفحة Deliver تتعامل مع التصدير بعشرات الإعدادات المسبقة للتنسيقات.

English: Take time to learn the layout of each page. The Edit page has the Media Pool (top left) for browsing clips, the Timeline (bottom) for sequencing, and the Viewer (top right) for preview. The Color page features the Gallery (stills for reference), Node Editor (where grading is built), and multiple scopes (waveform, vectorscope, histogram, parade). Customize your workspace by dragging panel dividers and saving layouts. Use the Workspace > Layout Presets to switch between editing and color grading layouts. Mastering navigation between these pages is the first step to efficient Resolve workflows.

العربية: خذ وقتك لتعلم تخطيط كل صفحة. صفحة Edit تحتوي على Media Pool (أعلى اليسار) لتصفح المقاطع، Timeline (أسفل) للتسلسل، و Viewer (أعلى اليمين) للمعاينة. صفحة Color تتميز بـ Gallery (لقطات ثابتة للمرجعية)، Node Editor (حيث يتم بناء التلوين)، وعدة scopes (موجات، vectorscope، رسم بياني، parade). خصص مساحة عملك بسحب فواصل اللوحات وحفظ التخطيطات. استخدم Workspace > Layout Presets للتبديل بين تخطيطات التحرير وتصحيح الألوان. إتقان التنقل بين هذه الصفحات هو الخطوة الأولى لسير عمل Resolve الفعال.

Setting Up a Project · إعداد المشروع

English: Start by creating a new project: File > New Project, give it a clear name. Set your timeline resolution and frame rate in Project Settings (gear icon at bottom right). Common settings: 1920x1080 at 23.976 fps for cinematic content, 1920x1080 at 29.97 fps for broadcast/web, 3840x2160 at 60 fps for high-frame-rate content. Set the timeline format to match your footage to avoid rendering issues. Enable Proxy Mode if working with 4K+ footage on a laptop — select "Quarter Resolution" or "Half Resolution" from the playback menu. In Project Settings > Image Scaling, set "Input Scaling" to "Scale Entire Image to Fit" and "Output Scaling" to match your delivery resolution.

العربية: ابدأ بإنشاء مشروع جديد: ملف > مشروع جديد، أعطه اسمًا واضحًا. اضبط دقة الخط الزمني ومعدل الإطارات في إعدادات المشروع (رمز الترس في أسفل اليمين). إعدادات شائعة: 1920x1080 بمعدل 23.976 إطارًا في الثانية للمحتوى السينمائي، 1920x1080 بمعدل 29.97 للبث/الويب، 3840x2160 بمعدل 60 للمحتوى عالي الإطارات. اضبط تنسيق الخط الزمني ليطابق لقطاتك لتجنب مشاكل التصيير. فعّل وضع Proxy إذا كنت تعمل بلقطات 4K+ على لابتوب — اختر "Quarter Resolution" أو "Half Resolution" من قائمة التشغيل.

English: Organize your Media Pool with bins (folders). Right-click in the Media Pool and create bins for: Footage (organized by date or scene), Audio (music, SFX, VO), Graphics (logos, titles, overlays), and Exports. Use Smart Bins to automatically collect clips by date, type, or metadata. Name your clips descriptively before dragging them to the timeline — this saves hours of searching later. Create timelines with naming conventions: "ProjectName_Cut_v01". Set your timeline to 30 fps for broadcast or 24 fps for cinema. Always save your project regularly — Resolve has an auto-save feature that creates backups, but manual saves after major changes are recommended.

العربية: نظّم Media Pool باستخدام الصناديق (مجلدات). انقر يمينًا في Media Pool وأنشئ صناديق لـ: Footage (منظم حسب التاريخ أو المشهد)، Audio (موسيقى، مؤثرات، تعليق صوتي)، Graphics (شعارات، عناوين، تراكبات)، و Exports. استخدم Smart Bins لجمع المقاطع تلقائيًا حسب التاريخ أو النوع. سمِّ مقاطعك بشكل وصفي قبل سحبها إلى الجدول الزمني — هذا يوفر ساعات من البحث لاحقًا. أنشئ جداول زمنية بتسمية موحدة: "اسم_المشروع_النسخة_v01". احفظ مشروعك بانتظام — Resolve لديه ميزة حفظ تلقائي، لكن الحفظ اليدوي بعد التغييرات الكبيرة موصى به.

Editing on the Cut & Edit Pages · التحرير في صفحات القص والتحرير

English: The Cut page is designed for speed. The Source Tape at the top plays all clips in your selected bin sequentially — mark In/Out points and append to the timeline with one click. The timeline shows two views: the top strip shows the full project duration for overview navigation, while the bottom strip shows detailed editing. Use the Trim mode for precise edge adjustments. The Cut page also includes a built-in Speed Editor hardware support for physical jog wheels and buttons. For more advanced editing, switch to the Edit page. Here you have full control: insert, overwrite, replace, and fit-to-fill edits. Use the Razor Edit mode (shortcut B) to split clips, or the Trim mode (T) for precision edits.

العربية: صفحة Cut مصممة للسرعة. Source Tape في الأعلى يشغل جميع المقاطع في الصندوق المحدد بالتسلسل — حدد نقاط In/Out وأضف إلى الجدول الزمني بنقرة واحدة. الجدول الزمني يظهر عرضين: الشريط العلوي يظهر مدة المشروع الكاملة للتنقل العام، بينما الشريط السفلي يظهر التحرير التفصيلي. استخدم وضع Trim لتعديلات الحواف الدقيقة. للتحرير المتقدم، انتقل إلى صفحة Edit. هنا لديك تحكم كامل: إدراج واستبدال واستبدال وملء. استخدم أداة Razor Edit (اختصار B) لتقسيم المقاطع، أو وضع Trim (T) للتعديلات الدقيقة.

English: Master the editing keyboard shortcuts: I and O for marking In and Out on source clips, F1-F10 for quick timeline navigation by timecode, Q and W for ripple trim from playhead to clip start/end, and Shift + / for adding edits to all tracks. The Source Browser (F9) lets you preview and mark clips before adding them. Use the Transform tool in the Inspector to scale, position, and rotate clips. The keyframe editor in the Inspector (diamond icon) tracks parameter changes over time. For multicam editing, select clips, right-click and choose "Create Multi-Cam from Clips" — sync by audio waveform or timecode and switch angles while playing.

العربية: أتقن اختصارات التحرير: I و O لتحديد In و Out على مقاطع المصدر، F1-F10 للتنقل السريع في الجدول الزمني، Q و W للقص المموج من المؤشر إلى بداية/نهاية المقطع، و Shift + / لإضافة قصات لجميع المسارات. Source Browser (F9) يتيح لك معاينة المقاطع قبل إضافتها. استخدم أداة Transform في Inspector لتغيير حجم وموضع وتدوير المقاطع. محرر الإطارات المفتاحية في Inspector (أيقونة الماسة) يتتبع تغييرات المعلمات عبر الوقت. للتحرير متعدد الكاميرات، اختر المقاطع وانقر يمينًا واختر "Create Multi-Cam from Clips" — وافق بالصوت أو timecode وبدّل الزوايا أثناء التشغيل.

Color Grading Workflow · سير عمل تصحيح الألوان

English: Resolve's Color page is world-renowned. The workflow follows a logical sequence. Step 1: Primary correction — balance exposure, set white balance, and restore contrast on each clip. Use the Color Wheels panel: Lift (shadows), Gamma (midtones), Gain (highlights), and Offset (master brightness). Adjust them so your waveform shows detail from 0 to 100 IRE with no clipping. Step 2: Match shots — use the HDR palette or auto-balance to match color and exposure across all clips in a scene. The Shot Match feature analyzes two clips and transfers the grade from one to the other automatically. Step 3: Creative grade — apply a look using the LUT Gallery or manually adjust the color wheels for a stylized feel.

العربية: صفحة Color في Resolve مشهورة عالميًا. سير العمل يتبع تسلسلًا منطقيًا. الخطوة 1: التصحيح الأساسي — وازن الإضاءة، اضبط توازن البياض، واستعد التباين على كل مقطع. استخدم لوحة Color Wheels: Lift (ظلال)، Gamma (درجات متوسطة)، Gain (إضاءات)، و Offset (سطوع رئيسي). اضبطها حتى يظهر ال waveform تفاصيل من 0 إلى 100 IRE بدون قص. الخطوة 2: طابق اللقطات — استخدم لوحة HDR أو التوازن التلقائي لمطابقة اللون والإضاءة عبر جميع اللقطات في المشهد. ميزة Shot Match تحلل لقطتين وتنقل التلوين من واحدة إلى الأخرى تلقائيًا. الخطوة 3: التلوين الإبداعي — طبق مظهرًا باستخدام LUT Gallery أو عدّل عجلات الألوان يدويًا لمظهر منمق.

English: Node trees are the heart of Resolve's color workflow. Each node holds a single grading operation. Build a node tree like this: Node 1: balance (exposure and white balance correction). Node 2: contrast (curves or contrast slider). Node 3: color matching (if working across multiple shots). Node 4: creative grade (look and style). Node 5: power windows (selective adjustments on specific areas). Node 6: grain and texture (film grain, sharpening, or softening). Right-click between nodes to add serial nodes, or Alt-click to add parallel nodes (for side-by-side operations like blending two grades). Label your nodes — double-click the node and type "WB", "Contrast", "Skin", etc. This makes complex grades navigable.

العربية: أشجار العقد هي قلب سير عمل الألوان في Resolve. كل عقدة تحمل عملية تلوين واحدة. ابنِ شجرة عقد مثل هذه: العقدة 1: التوازن (تصحيح الإضاءة وتوازن البياض). العقدة 2: التباين (منحنيات أو شريط التباين). العقدة 3: مطابقة الألوان (إذا كنت تعمل عبر لقطات متعددة). العقدة 4: التلوين الإبداعي (المظهر والأسلوب). العقدة 5: النوافذ القوية (تعديلات انتقائية على مناطق محددة). العقدة 6: الحبيبات والملمس (حبيبات فيلم، توضيح، أو تليين). انقر يمينًا بين العقد لإضافة عقد تسلسلية، أو Alt+نقر لإضافة عقد متوازية. سمِّ عقدك — انقر مرتين على العقدة واكتب "WB" أو "تباين" أو "بشرة". هذا يجعل التلوين المعقد قابلاً للتنقل.

Fusion Visual Effects · مؤثرات فيوجن البصرية

English: Fusion is a node-based compositor built into Resolve. Unlike layer-based systems (like After Effects), each node performs exactly one operation, connected by lines to other nodes. The basic flow is: MediaIn (your footage) connects to a series of effect nodes, then to MediaOut (the result). Common nodes: Merge (combines two images — foreground over background), Transform (move, scale, rotate), Text+ (titles and typography with full animation controls), Background (solid colors, gradients), Blur (Gaussian, directional, radial), and ColorCorrector (adjust color in Fusion without going to the Color page).

العربية: فيوجن هو نظام تركيب قائم على العقد مبني داخل Resolve. على عكس الأنظمة القائمة على الطبقات، كل عقدة تؤدي عملية واحدة بالضبط، متصلة بخطوط بعقد أخرى. التدفق الأساسي هو: MediaIn (لقطاتك) يتصل بسلسلة من عقد التأثيرات، ثم إلى MediaOut (النتيجة). العقد الشائعة: Merge (يدمج صورتين — المقدمة فوق الخلفية)، Transform (تحريك، تحجيم، تدوير)، Text+ (عناوين وطباعة مع تحكم كامل بالتحريك)، Background (ألوان صلبة، تدرجات)، Blur (Gaussian، اتجاهي، شعاعي)، و ColorCorrector (ضبط الألوان في Fusion دون الذهاب إلى صفحة Color).

English: To create a title animation in Fusion: add a Text+ node after MediaIn. Type your text, choose a font (use Cairo or Tajawal for Arabic), and set size and alignment. Add a Merge node to composite the text over your footage. Right-click the text node's Transform tab — select "Animate" to enable keyframing. Set position keyframes to animate the text on screen. Use the Spline view (Shift+Space, type "Spline") to adjust animation curves for smooth motion — the curve editor in Fusion is similar to After Effects' Graph Editor. For particles, use the pEmitter and pRender nodes — they create rain, snow, fire, smoke, or abstract particle backgrounds. Fusion is extremely powerful but has a learning curve — start with simple text animations and compound nodes.

العربية: لإنشاء تحريك عنوان في Fusion: أضف عقدة Text+ بعد MediaIn. اكتب نصك، اختر خطًا (استخدم Cairo أو Tajawal للعربية)، واضبط الحجم والمحاذاة. أضف عقدة Merge لدمج النص فوق لقطاتك. انقر يمينًا على تبويب Transform لعقدة النص — اختر "Animate" لتفعيل الإطارات المفتاحية. اضبط إطارات مفتاحية للموضع لتحريك النص على الشاشة. استخدم Spline view لضبط منحنيات التحريك للحركة الناعمة. للجسيمات، استخدم عقد pEmitter و pRender — تنشئ مطرًا وثلجًا ونارًا ودخانًا أو خلفيات جسيمات تجريدية. فيوجن قوي جدًا لكن له منحنى تعليمي — ابدأ بتحريك النص البسيط والعقد المركبة.

Fairlight Audio · صوت فيرلايت

English: Fairlight is a full-featured digital audio workstation within Resolve. Switch to the Fairlight page for serious audio work. The page features a multitrack timeline with advanced routing, built-in effects (EQ, compressor, limiter, reverb, noise reduction), and VST3 plugin support for third-party audio tools. Each track has a mixer strip with volume fader, pan control, and mute/solo buttons. Use the Fairlight Audio Assistant (select clips, right-click > Fairlight Audio Assistant) for automatic dialogue leveling and clean-up. The built-in Dialogue Leveler uses adaptive compression to keep speech at a consistent volume.

العربية: فيرلايت هي محطة عمل صوتية رقمية كاملة داخل Resolve. انتقل إلى صفحة Fairlight لأعمال الصوت الجادة. الصفحة تتميز بخط زمني متعدد المسارات مع توجيه متقدم وتأثيرات مدمجة ودعم ملحقات VST3. كل مسار لديه شريط خلاط بمقياس صوت وتحكم بانوراما وأزرار كتم/انفراد. استخدم Fairlight Audio Assistant (حدد المقاطع، انقر يمينًا > Fairlight Audio Assistant) لتسوية الحوار التلقائية والتنظيف.

English: For dialogue cleanup: add a DeNoiser effect — it adaptively removes background hum, fan noise, and air conditioning. The EQ effect shapes the frequency spectrum: cut below 80Hz for rumble, boost 2-4kHz for clarity, and cut 8kHz+ for sibilance. Use the Compressor to even out volume variations — set Ratio 3:1, Attack 10ms, Release 100ms, and adjust Threshold until compression is 3-6dB on peaks. For background music, lower the volume to -18dB to -12dB and use sidechain compression to duck music automatically when dialogue is present. Add reverb to voiceovers for a sense of space — a small room reverb (0.3s decay, 15% wet) adds warmth without sounding echoey.

العربية: لتنظيف الحوار: أضف تأثير DeNoiser — يزيل ضوضاء الخلفية بشكل تكيفي. تأثير EQ يشكل طيف الترددات: اقطع تحت 80Hz للدمدمة، عزز 2-4kHz للوضوح، اقطع 8kHz+ للصفير. استخدم الضاغط لتسوية تغيرات الصوت — اضبط Ratio 3:1، Attack 10ms، Release 100ms، وعدّل Threshold حتى يكون الضغط 3-6dB على الذروات. للموسيقى الخلفية، اخفض الصوت إلى -18dB إلى -12dB واستخدم الضغط الجانبي لخفض الموسيقى تلقائيًا عند وجود حوار. أضف reverberation للتعليق الصوتي لإحساس بالمساحة — reverberation غرفة صغيرة يضيف دفئًا دون أن يبدو صدى.

Export Settings · إعدادات التصدير

English: The Deliver page provides all export options organized by platform preset. For YouTube/Vimeo: choose the H.264 preset under "Video Sharing". Set resolution to 3840x2160 (4K) or 1920x1080 (HD). Enable "Render at Maximum Size" for best quality. Set bitrate to 16-20 Mbps for 1080p or 40-60 Mbps for 4K using VBR 2-Pass (Variable Bit Rate, two passes for maximum quality). Always select "Render at Maximum Depth" (10-bit) if your footage is 10-bit. For social media (Instagram Reels, TikTok): choose the H.264 Master preset, set resolution to 1080x1920 (portrait 9:16), and bitrate to 10-15 Mbps. For client delivery: use the "Apple ProRes 422 HQ" preset for Mac workflows or "DNxHR HQ" for Windows.

العربية: صفحة Deliver توفر كل خيارات التصدير منظمة حسب إعدادات المنصة المسبقة. لـ YouTube/Vimeo: اختر إعداد H.264 المسبق تحت "Video Sharing". اضبط الدقة على 4K أو HD. فعّل "Render at Maximum Size" لأفضل جودة. اضبط معدل البت على 16-20 Mbps لـ 1080p أو 40-60 Mbps لـ 4K باستخدام VBR 2-Pass. دائمًا اختر "Render at Maximum Depth" إذا كانت لقطاتك 10-bit. لوسائل التواصل (Instagram Reels، TikTok): اختر إعداد H.264 Master، اضبط الدقة على 1080x1920 (عمودي 9:16)، ومعدل البت 10-15 Mbps. للتسليم للعميل: استخدم "Apple ProRes 422 HQ" لـ Mac أو "DNxHR HQ" لـ Windows.

English: Advanced export tips: Add 2 seconds of black at the end for streaming platforms. Use "Individual Clips" mode in the Deliver page to export timelines as separate files per marker — perfect for creating chapters or social media cuts. Enable "Force Debayer to Highest Quality" for maximum sharpness on BMD RAW footage. Set audio to AAC format, 320 kbps, Stereo. For archiving masters, use "Uncompressed" or "ProRes 4444" formats. Use the Render Queue (Add to Render Queue) to queue multiple exports and continue working. Resolve can also export individual frames — right-click the viewer and choose "Export Frame" for high-quality stills. Always spot-check your export by viewing on the target platform.

العربية: نصائح تصدير متقدمة: أضف ثانيتين من الأسود في النهاية لمنصات البث. استخدم وضع "Individual Clips" في صفحة Deliver لتصدير الجداول الزمنية كملفات منفصلة حسب العلامة — مثالية لإنشاء فصول أو مقاطع وسائل التواصل. فعّل "Force Debayer to Highest Quality" لأقصى وضوح على لقطات BMD RAW. اضبط الصوت على AAC، 320 kbps، ستيريو. للأرشفة، استخدم صيغ "Uncompressed" أو "ProRes 4444". استخدم Render Queue (أضف إلى قائمة التصيير) لجدولة تصديرات متعددة ومواصلة العمل. يمكن لـ Resolve أيضًا تصدير إطارات فردية — انقر يمينًا على الشاشة واختر "Export Frame" للقطات ثابتة عالية الجودة.

Keyboard Shortcuts · اختصارات لوحة المفاتيح

English: Speed in Resolve comes from keyboard shortcuts. Here are the essential ones organized by page. Cut/Edit page: I/O (mark In/Out), F9 (Source Browser), F10 (Media Pool), F11 (Timeline), F12 (Inspector), Ctrl+I (import media), Ctrl+S (save), Ctrl+Z (undo), Ctrl+Shift+Z (redo), Ctrl+X/C/V (cut/copy/paste). Trim mode: W (ripple trim previous clip's end), Q (ripple trim next clip's start), A (select clips), B (razor edit), T (trim mode). Color page: Alt+S (add serial node), Alt+P (add parallel node), Alt+L (add layer node), D (disable node), M (toggle full-screen monitor), Shift+, / Shift+. Project Management · إدارة المشاريع

English: Resolve offers robust project management. Use the Project Manager (File > Project Manager) to organize, back up, and archive projects. Each project stores its own database — you can clone a project for experimenting, export as a .drp archive for sharing, or consolidate media for archiving. The Media Management tool (File > Media Management) copies or relinks footage to a new location, trimming unused portions to save space. Use this when finalizing a project — it creates a self-contained folder with only the footage you used. For collaborative workflows, enable Collaboration (Workspace > Collaboration) to allow multiple editors to work on the same project in real-time. Each user gets their own timeline and bins, with changes synced automatically over the network.

العربية: Resolve يقدم إدارة مشاريع قوية. استخدم Project Manager (ملف > مدير المشروع) لتنظيم ونسخ احتياطي وأرشفة المشاريع. كل مشروع يخزن قاعدة بياناته الخاصة — يمكنك استنساخ مشروع للتجربة، تصدير كأرشيف .drp للمشاركة، أو دمج الوسائط للأرشفة. أداة Media Management (ملف > إدارة الوسائط) تنسخ أو تعيد ربط اللقطات لموقع جديد، مقلمة الأجزاء غير المستخدمة لتوفير المساحة. استخدمها عند إنهاء مشروع — تنشئ مجلدًا مكتفيًا ذاتيًا باللقطات التي استخدمتها فقط. لسير العمل التعاوني، فعّل Collaboration (مساحة العمل > تعاون) للسماح لمحررين متعددين بالعمل على نفس المشروع في الوقت الفعلي.

Conclusion · خلاصة

English: DaVinci Resolve is a complete post-production powerhouse. Its integration of editing, color, VFX, audio, and delivery in a single application is unmatched in the industry. The free version gives you 90% of the professional capabilities — there is no better value in video editing. Start by learning the Cut page for fast editing, then graduate to the Edit page for more control. Invest time in understanding the Color page — it is what sets Resolve apart. Experiment with Fusion for motion graphics, and explore Fairlight for audio post-production. Watch tutorials, practice on real projects, and build your skills one page at a time. Resolve has a steep learning curve, but every hour you invest returns tenfold in editing power and creative possibilities.

Resolve له منحنى تعليمي حاد، لكن كل ساعة تستثمرها تعود بعشرة أضعاف في قوة التحرير والإمكانيات الإبداعية.
